## 主要问题分析

### 1. 前端服务未启动 ⚠️

**错误**: `net::ERR_INSUFFICIENT_RESOURCES at http://localhost:5173/`

**原因**: Playwright 配置中设置了自动启动前端服务，但可能启动失败或端口被占用。

**解决方案**:
```bash
# 手动启动前端（推荐）
cd frontend
npm run dev

# 然后在另一个终端运行测试
npm run test:e2e
```

### 2. 选择器严格模式冲突

**错误**: `strict mode violation: getByRole('link', { name: /职位/i }) resolved to 8 elements`

**原因**: 页面上有多个匹配 "职位" 的链接，Playwright 不知道选哪个。

**影响**: 导航测试失败

**修复方案**: 需要更精确的选择器（我可以帮你修复）

### 3. 后端服务可能未启动

很多测试失败可能是因为 API 请求失败。
