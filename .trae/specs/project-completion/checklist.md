# 启航平台全系统联调与完善 - 验证清单

## 一、🔴 基础环境联调验收（必须全部通过）

### 1.1 后端服务启动 \[Task 1]

* [ ] backend/.env 文件存在且配置完整（对比 .env.example 无缺失项）

* [ ] JWT\_SECRET 已配置为 64 位随机字符串（非默认值）

* [ ] JWT\_REFRESH\_SECRET 已配置为 64 位随机字符串（非默认值）

* [ ] ENCRYPTION\_KEY 已配置为 64 位十六进制（或明确不需要）

* [ ] 数据库连接信息正确（DB\_HOST、DB\_PORT、DB\_USER、DB\_PASSWORD、DB\_NAME）

* [ ] `node server.js` 启动成功，控制台无 FATAL 错误

* [ ] 控制台显示「数据库连接成功」和「服务运行在端口 XXXX」

### 1.2 前端代理与 API 连通 \[Task 2]

* [ ] frontend/vite.config.ts 中 proxy 配置指向后端正确端口

* [ ] frontend/.env 中 VITE\_API\_URL 配置正确

* [ ] 前端启动后（npm run dev）控制台无代理相关错误

* [ ] 浏览器访问前端页面，Network 面板中 API 请求状态码非 0 或 (failed)

* [ ] 登录接口返回有效响应（成功或预期的错误格式）

### 1.3 数据库与种子数据 \[Task 3]

* [ ] init-db.js 执行成功，所有表创建无误

* [ ] 数据库包含管理员账户（用户名/密码可登录）

* [ ] 种子数据包含：演示用户（≥5个）、企业（≥3个）、导师（≥3个）、课程（≥5个）、职位（≥5个）

* [ ] 管理员登录后仪表盘显示有数据（非全零或空白）
