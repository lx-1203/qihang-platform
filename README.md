# 启航平台 - 大学生综合发展与就业指导平台

为大学生提供求职、考研、留学、创业全方位指导的一站式平台。

## 项目信息

| 属性       | 内容                                                   |
| ---------- | ------------------------------------------------------ |
| 项目类型   | B/S 架构 Web 应用（前后端分离）                        |
| 服务对象   | 在校大学生、应届毕业生、企业招聘方、职业导师            |
| 立项单位   | 江苏初晓云网络科技有限公司技术组                       |
| 项目状态   | 前端 UI 骨架搭建完成，后端待搭建                       |

## 功能模块

| 角色         | 核心功能                                                 |
| ------------ | -------------------------------------------------------- |
| **学生端**   | 职位浏览与投递、课程学习、导师预约、收藏管理、个人档案   |
| **企业端**   | 职位发布与管理、简历筛选(Kanban)、人才搜索、数据统计     |
| **导师端**   | 课程创建与管理、预约管理、学生管理、数据看板             |
| **管理后台** | 用户管理、企业/导师审核、内容管理、平台统计              |

## 技术栈

**前端**：React 19 + TypeScript + Vite + Tailwind CSS + Framer Motion + Zustand + Axios + i18next

**后端**：Express + MySQL 8.0 + JWT + bcryptjs + Multer

## 快速开始

### 环境要求

- Node.js 18.x 或 20.x
- MySQL 8.0+

### 安装与运行

```bash
# 1. 克隆项目
git clone https://github.com/lx-1203/qihang-platform.git
cd qihang-platform

# 2. 安装前端依赖
cd frontend
npm install

# 3. 启动前端开发服务器
npm run dev
```

启动后访问 `http://localhost:5173`

> 后端服务搭建中，完成后将支持全栈启动（`start.bat` / `start.sh`）。
