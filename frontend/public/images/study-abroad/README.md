# 留学板块图片资源

所有图片都来自 Unsplash，可免费使用。

## 图片使用方式

### 方式1：直接使用 Unsplash URL（推荐）
在代码中直接引用下方配置文件中的 URL。

### 方式2：下载到本地
如需下载，请访问下方链接手动保存：
- 右键 → 图片另存为 → 保存到 `public/images/study-abroad/` 文件夹

## 图片文件夹结构

```
public/images/study-abroad/
├── hero/              # Hero轮播图
├── services/          # 服务卡片图
├── stories/           # 学员故事图
├── quotes/            # 新人寄语背景图
└── README.md          # 本文件
```
