# 无代码配置指南

## 🎉 无需懂代码，3步修改留学板块！

### 第一步：打开配置文件

在项目中找到这个文件：
```
frontend/src/data/study-abroad-ui-config.json
```

用记事本、VS Code 或任何文本编辑器打开它。

---

### 第二步：修改你想改的内容

#### 修改 Hero 轮播图

找到 `heroSlides` 这一段，修改里面的文字：

```json
{
  "title": "这里改标题",
  "subtitle": "这里改副标题",
  "image": "https://... 这里改图片链接",
  "cta": "这里改按钮文字"
}
```

#### 修改服务卡片

找到 `serviceCards`，修改8个卡片的内容：

```json
{
  "title": "改卡片标题",
  "subtitle": "改副标题",
  "description": "改描述",
  "image": "https://... 改图片"
}
```

#### 修改学员故事

找到 `studentStories`，修改故事内容：

```json
{
  "studentName": "改名字",
  "background": "改背景",
  "result": "改录取结果",
  "quote": "改金句",
  "story": "改完整故事",
  "image": "https://... 改图片"
}
```

#### 修改新人寄语

找到 `newcomerQuotes`，修改语录：

```json
{
  "quote": "改语录",
  "author": "改作者",
  "background": "改背景介绍"
}
```

---

### 第三步：如何换图片？

#### 方式A：直接用 Unsplash 链接（最简单）

1. 访问 https://unsplash.com
2. 搜索你想要的图片（比如 "international students"）
3. 点击图片 → 点击 "Download Free" 旁边的箭头
4. 复制图片地址（右键 → 复制图片地址）
5. 粘贴到配置文件的 `image` 字段

#### 方式B：用本地图片

1. 把图片保存到这个文件夹：
   ```
   frontend/public/images/study-abroad/
   ```

2. 在配置文件中这样写：
   ```json
   "image": "/images/study-abroad/你的图片文件名.jpg"
   ```
